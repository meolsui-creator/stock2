삼성전자: 종목코드 005930, DART corp_code = 00126380.

라벨 특이사항:
- 2023 사업보고서(2024.03 접수) 연결 손익계산서의 매출액은 `매출액`이 아니라 `영업수익` 계정명으로 태깅됨. 2024·2025는 `매출액`.
- 사업보고서 접수 시기: 3월 초~중순(2024.12분 → 2025.03.11, 2025.12분 → 2026.03.10).
- 분기보고서 접수: 1Q 5월 중순, 반기 8월 중순, 3Q 11월 중순.

검증된 연간 연결(CFS) 수치 (단위 원):
- 2023: 매출 258,935,494백만원(258.9조)/영익 6,566,976백만/순익 15,487,100백만
- 2024: 매출 300,870,903백만/영익 32,725,961백만/순익 34,451,351백만
- 2025: 매출 333,605,938백만/영익 43,601,051백만/순익 45,206,805백만

**데이터 이상(주의):** 1Q2026(제58기 1분기, 2026.05.15 접수) 분기보고서의 손익계산서가
DART OpenAPI에서 비정상적으로 큰 값으로 태깅됨(연결 매출 133.8조/영익 57.2조, 별도 매출 109.3조/영익 49.2조).
이는 연 매출 333.6조 대비 분기값으로 비현실적. frmtrm(전기 1Q2025)도 매출 79.1조/영익 6.7조로 동일 이상.
**How to apply:** 1Q2026 분기 실적 및 그로부터 파생되는 QoQ/YoY는 원천 데이터 신뢰 불가 → "확인 불가"로 처리.
재요청 시 분기 IS는 단순 채택하지 말고 회사 IR/원문 대조 필요.
