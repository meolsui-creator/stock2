# report-pptx

가치투자 리서치 마크다운(`reports/{종목명}.md`)을 **투자증권 리서치센터 톤의 PPTX**로 변환합니다.

## 실행 방법

```bash
python .claude/skills/report-pptx/build_pptx.py "<종목명>"
```

- 입력: `reports/{종목명}.md`
- 출력: `reports/pptx/{종목명}.pptx` (`pptx` 폴더가 없으면 자동 생성)
- 의존성: `python-pptx` (없으면 `python -m pip install python-pptx`)
- 현재 폴더가 프로젝트 루트가 아니면 `--root "<프로젝트경로>"`를 붙입니다.

## 슬라이드 순서 (항상 고정)

1. **표지** — 종목명 + 작성일
2. **종목 개요**
3. **재무 요약** — 표(최근 3개년)
4. **가격/추세**
5. **뉴스·심리**
6. **리스크**
7. **한 줄 종합**

## 입력 .md 작성 규칙 (파서 계약)

스크립트는 아래 구조를 파싱합니다. 섹션 제목은 키워드로 유연하게 매칭됩니다.

```markdown
# 삼성전자
작성일: 2026-06-16

## 종목 개요
- KOSPI 대형주 ... (출처/기준일)

## 재무 요약
| 항목 | 2023 | 2024 | 2025 |
|---|---|---|---|
| 매출액(조원) | 258.9 | 300.9 | 333.6 |
| 영업이익(조원) | 6.6 | 32.7 | 43.6 |
(출처: DART 전자공시 / 기준일 2026-06-16, 연결)

## 가격/추세
- ... (출처/기준일)

## 뉴스·심리
- ... (출처/기준일)

## 리스크
- ... (출처/기준일)

## 한 줄 종합
밸류에이션·수익성 회복 근거를 종합하면 ...입니다.
```

- 제목 행: `# 종목명` (첫 H1)
- 작성일: `작성일: YYYY-MM-DD` (없으면 오늘 날짜로 자동 채움)
- 각 슬라이드 본문: `## 제목` 아래의 표/불릿/문장
- **재무 요약**의 표는 마크다운 표(첫 행이 헤더)로 작성하면 리서치 표 양식으로 렌더링됩니다.
- 키워드 매칭: 개요·소개·기업 / 재무·실적 / 가격·추세·차트·기술 / 뉴스·심리·센티·이슈 / 리스크·위험 / 종합·결론·한 줄
- 섹션이 없으면 해당 슬라이드는 "출처 있는 내용만 채웁니다" 안내로 비워 둡니다.

## 디자인 사양 (코드에 고정)

- 배경 화이트 / 제목 **딥네이비 `#1F2A44`** / 본문 **그레이**
- **한화 오렌지 `#F37321`**: 표지 상단 바·제목 아래 구분선·키 강조에만 절제 사용
- 표: 헤더행만 딥네이비 배경 + 흰 글자, 본문행은 얇은 그레이 괘선
- 한글 폰트는 **맑은 고딕** 하나로 고정 (라틴/동아시아 폰트 동일 지정으로 깨짐 방지)

## 가드레일 (CLAUDE.md 준수)

- 모든 수치 옆 **(출처/기준일)** 표기 — 출처 없는 숫자는 .md에 넣지 않습니다.
- **매수/매도·목표가 단정 금지** — 스크립트는 판단 문구를 만들지 않으며, 모든 슬라이드 하단에 "학습용 분석 자료 · 매수/매도·목표가 판단을 제공하지 않습니다"를 표기합니다.
- 본 산출물은 **학습용 분석**임을 명시합니다.
