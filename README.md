# 📈 stock2 — 종목 분석 리포트 자동화 (실습용)

> 종목 이름만 알려주면 **AI 애널리스트들이 협업**해서 재무·차트·뉴스·리스크를
> 분석하고, **증권사 리서치센터 스타일의 PPTX 리포트**까지 만들어 주는 실습용 프로젝트입니다.
>
> ⚠️ **학습용 더미 프로젝트입니다.** 실제 투자 판단(매수/매도)에 사용하지 마세요.

이 README는 **컴퓨터·코딩이 처음인 분**도 따라 할 수 있도록
"이 버튼을 누르세요" 수준으로 아주 자세히 설명합니다. 천천히 따라오세요. 🙂

---

## 1. 이게 뭐 하는 거예요? (3줄 요약)

1. `삼성전자` 같은 **종목 이름**을 입력합니다.
2. 4명의 AI 애널리스트가 **재무 / 차트 / 뉴스·심리 / 리스크**를 나눠서 분석합니다.
3. 결과를 모아 **마크다운 리포트 + 발표용 PPTX/HTML 슬라이드**로 내보냅니다.

분석은 **가치투자 관점**으로 하며, 모든 숫자에는 **출처와 기준일**을 붙입니다.

---

## 2. 미리 준비할 것 (준비물)

이 프로젝트는 **Claude Code**라는 AI 도구 안에서 돌아갑니다. 아래 3가지가 필요합니다.

| 준비물 | 무엇인지 | 어디서 얻나요 |
|---|---|---|
| **Claude Code** | AI에게 일을 시키는 프로그램 (이 프로젝트의 핵심) | <https://claude.com/claude-code> |
| **Python 3** | 슬라이드(PPTX/HTML)를 만드는 데 사용 | <https://www.python.org/downloads/> 설치 시 "Add Python to PATH" 체크 ✅ |
| **DART API 키** | 재무제표를 가져오는 무료 열쇠 (재무 분석에만 필요) | <https://opendart.fss.or.kr> 회원가입 → 인증키 신청 (무료, 즉시 발급) |

> 💡 **뉴스·차트 분석만** 할 거라면 DART 키는 없어도 됩니다. 재무 분석을 하려면 꼭 필요합니다.

## 3. 설치하기 (딱 한 번만 하면 됩니다)

### 3-1. 이 프로젝트 내려받기

컴퓨터의 명령어 창(Windows는 **PowerShell**)을 열고 아래를 한 줄씩 입력하세요.

```bash
git clone https://github.com/meolsui-creator/stock2.git
cd stock2
```

> `git`이 없다고 나오면 <https://git-scm.com> 에서 먼저 설치하세요.

### 3-2. DART 키 넣기 (재무 분석을 할 경우만)

프로젝트 폴더 안에 **`.env`** 라는 이름의 파일을 새로 만들고, 아래 한 줄을 적습니다.
`여기에_발급받은_키` 자리에 본인이 OpenDART에서 받은 키를 붙여넣으세요.

```
DART_KEY=여기에_발급받은_키
```

> 🔒 `.env` 파일은 **절대 GitHub에 올라가지 않습니다** (`.gitignore`로 막아둠).
> 본인의 키는 본인만 갖고 있어야 합니다.

### 3-3. 슬라이드 만들기용 라이브러리 설치

PPTX 리포트를 만들려면 한 번만 아래를 실행하세요.

```bash
python -m pip install python-pptx
```

> HTML 마켓 아웃룩 덱만 만들 거라면 이 설치도 필요 없습니다 (표준 기능만 사용).

## 4. 사용하기 (가장 쉬운 방법)

이 프로젝트는 **Claude Code에게 한국어로 말만 걸면** 알아서 일합니다.
폴더 안에서 Claude Code를 켜고 아래처럼 입력해 보세요.

| 하고 싶은 일 | 이렇게 말하세요 |
|---|---|
| 전체 리포트 한 번에 | `삼성전자 리포트 만들어줘` |
| 재무만 | `삼성전자 재무 분석해줘` |
| 차트만 | `삼성전자 차트 봐줘` |
| 뉴스·심리만 | `삼성전자 최근 이슈 정리해줘` |
| PPTX로 내보내기 | `방금 리포트 PPTX로 만들어줘` |

그러면 Claude가 적절한 애널리스트를 골라 분석하고, 결과를 `reports/` 폴더에 저장합니다.

### 직접 슬라이드만 만들고 싶다면 (명령어)

리포트 마크다운(`reports/종목명.md`)이 이미 있다면, 슬라이드는 직접 만들 수도 있습니다.

```bash
# 종목 리포트 → PPTX (reports/pptx/삼성전자.pptx 생성)
python .claude/skills/report-pptx/build_pptx.py "삼성전자"

# 마켓 아웃룩 발표 덱 → HTML (reports/decks/ 에 생성)
python .claude/skills/outlook-deck/build_deck.py --cover A
```

## 5. 결과물은 어디에 생기나요?

모든 산출물은 **`reports/` 폴더**에 저장됩니다.

```
reports/
├── 삼성전자.md            ← 리서치 본문 (마크다운)
├── pptx/삼성전자.pptx     ← 발표용 PPT 리포트
├── decks/삼성전자.html    ← 웹 슬라이드
└── charts/삼성전자_price.png  ← 주가 차트 이미지
```

> 📂 저장소에는 **`삼성전자` 예시 결과물**이 들어 있으니, 먼저 열어보면 어떤 결과가 나오는지 감이 옵니다.

## 6. 누가 무슨 일을 하나요? (AI 애널리스트 4명)

| 애널리스트 | 담당 | 데이터 출처 | DART 키 필요? |
|---|---|---|---|
| `fundamental-analyst` | ② 재무·실적·공시 | DART OpenAPI | ✅ 필요 |
| `market-technical-analyst` | ③ 주가·추세·거래량 | FinanceDataReader | ❌ 불필요 |

리포트는 항상 **5부 구성**입니다: ① 표지 ② 재무 ③ 차트 ④ 리스크 ⑤ 종합의견

## 7. 잘 안 될 때 (자주 묻는 문제)

| 증상 | 해결 방법 |
|---|---|
| `python을 찾을 수 없습니다` | Python을 설치하고, 설치 시 **"Add to PATH"** 를 체크했는지 확인 |
| `No module named 'pptx'` | `python -m pip install python-pptx` 다시 실행 |
| 재무 분석이 `DART_KEY 미설정`이라고 함 | `.env` 파일에 `DART_KEY=...` 가 제대로 적혔는지 확인 |
| `git을 찾을 수 없습니다` | <https://git-scm.com> 에서 git 설치 |
